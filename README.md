# Description générale du répertoire

Conformément aux exigences de l'évaluation, ce dossier contient le **fichier XML-TEI d'origine ('nicolay_testaments.xml')** ainsi que la **feuille de style XSL ('transformation_html.xsl')**. 

L'objectif de ce projet est de créer, par le biais d'une transformation XSLT, un **site constitué de 5 fichiers HTML (home, index, pages 1/2/3)**. Ces fichiers sont regroupés automatiquement dans un dossier intitulé 'html', et sont liés lors de leur création à une **feuille de style 'nicolay.css'** disponible dans le dossier 'css' de ce répertoire. Il n'y a donc pas besoin de déplacer de fichiers pour faire fonctionner le code après la transformation.

Par souci d'exhaustivité, le dossier 'html' a été inclu dans le répertoire, mais il n'est pas nécessaire et les fichiers contenus seront réécrits dès le lancement de la transformation.
