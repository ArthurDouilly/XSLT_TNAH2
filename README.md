# Description générale du répertoire

Conformément aux exigences de l'évaluation, ce dossier contient le fichier XML-TEI d'origine ('nicolay_testaments.xml') ainsi que la feuille de style XSL ('transformation_html.xsl').
L'objectif de ce projet est de créer, par le biais d'une transformation XSLT, un site constitué de 5 fichiers HTML (home, index, pages 1/2/3), lesquelles sont associées à la feuille de style CSS ci-jointe ('nicolay.css').
