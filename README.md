# Description générale du répertoire

## Objectif de la transformation XSLT

L'objectif de ce projet est de créer, par le biais d'une transformation XSLT, un 'site internet' constitué de fichiers HTML valides selon le standard HTML et liés entre eux par des hyperliens. 

## Contenus du répertoire

Conformément aux exigences de l'évaluation, ce dossier contient le fichier XML-TEI d'origine ('nicolay_testaments.xml') ainsi que la feuille de style XSL ('transformation_html.xsl'). 

Il contient également les fichiers HTML (home, index, pages 1/2/3) résultant de la transformation XSLT. Les fichiers créés par la transformation sont regroupés automatiquement dans un dossier intitulé 'html', et sont liés lors de leur création à une feuille de style 'nicolay.css' disponible dans le dossier 'css' de ce répertoire. Il n'y a donc pas besoin de déplacer de fichiers pour faire fonctionner le code après la transformation.
